Detailed Description

BPSK (Binary Phase Shift Keying) modulation scheme policy struct.

Implements the modulation policy interface required by LinkSimulation. Maps each bit to a complex symbol on the I axis — bit 0 to {+1, 0} and bit 1 to {-1, 0} — representing 0° and 180° phase states of the carrier respectively. Q component is always zero for pure BPSK.

Note
Intended for use as a template parameter: LinkSimulation<BPSK>

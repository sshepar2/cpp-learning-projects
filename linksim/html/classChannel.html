Detailed Description

Models a noisy transmission channel.

Adds complex Gaussian noise to IQ symbols to simulate transmission impairments. The noise power is determined by the SNR per symbol passed to transmit(), allowing the same Channel instance to be reused across multiple SNR values in a sweep.

Currently supports AWGN (Additive White Gaussian Noise) only. Independent Gaussian noise is added to both I and Q components.

SimConfig parseConfig ( const std::string &  path)

Parses a JSON configuration file and returns a SimConfig struct.

Opens and parses the specified JSON file, populates a SimConfig struct, and validates all fields before returning. Throws on missing keys, malformed JSON, or invalid parameter values.

Expected JSON structure:

{
"modulation": "BPSK",
"num_bits": 10000,
"snr_range_db": { "min": -5.0, "max": 15.0, "step": 1.0 },
"noise_model": "AWGN",
"seed": 42,
"output_file": "results/ber_results.csv"
}
Parameters
pathPath to the JSON config file
Returns
Populated and validated SimConfig struct
Exceptions
std::runtime_errorif the file cannot be opened
std::invalid_argumentif any field fails validation
Note
snr sweep will start at "min" and stop at "max" thus "step" > 0
The "seed" field is optional. If omitted, a hardware random seed is used.

Writes a BER curve to a CSV file.

Outputs a two-column CSV with a header row, one row per SNR point. Values are written in fixed precision to six decimal places. Any directories in the CSV path which do not exist prior to calling will be created by this function.

Output format:

snr_db,ber
-5.000000,0.152341
0.000000,0.078432
Parameters
pathPath to the output CSV file
berCurveVector of {SNR, BER} pairs to write
Exceptions
std::runtime_errorif the file cannot be opened
Precondition
berCurve must not be empty

Detailed Description

Configuration parameters for a link simulation run.

Populated by parseing a JSON config file. Controls modulation scheme, bit count, SNR sweep range, noise model, and output destination. SNR is the SNR ber bit (Eb/N0), which is the energy per bit to noise power spectral density ratio.
